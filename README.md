# Diwali_sales_analysis

Diwali Sales Analysis (Python)

This project focuses on analyzing Diwali sales data using Python to
understand customer purchasing behavior and sales trends.

The analysis is performed using popular data science libraries and
covers data cleaning, exploratory data analysis (EDA), and basic
visualization to extract meaningful insights from the dataset.

--------------------------------------------------

Objectives
- Analyze customer demographics and buying patterns
- Identify top-selling product categories
- Understand sales trends during the Diwali season
- Gain hands-on experience with real-world data analysis

--------------------------------------------------

Tools & Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

--------------------------------------------------

Key Steps Performed
- Data loading and inspection
- Data cleaning and preprocessing
- Handling missing and duplicate values
- Exploratory Data Analysis (EDA)
- Visualization of sales trends and customer behavior

--------------------------------------------------

Skills Gained
- Data manipulation using Pandas
- Numerical operations using NumPy
- Data visualization techniques
- Analytical thinking and interpretation of results

--------------------------------------------------

Conclusion
This project helped in developing a practical understanding of data
analysis workflow and provided exposure to handling and analyzing
real-world sales data using Python.
